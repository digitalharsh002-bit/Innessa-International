Place any local product images here.

The project currently uses Unsplash URLs for all product images.
To use local images instead, update the image paths in lib/data.ts.

Recommended image specs:
- Product images: 600x800px (3:4 ratio)
- Category images: 400x400px (1:1 ratio)
- Hero poster: 1920x1080px
- About/banner images: 1400x600px

Image formats: WebP (preferred), JPG, PNG
